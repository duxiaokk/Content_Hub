演示说明

以下展示了平台从投递任务到获取结果的完整过程。

Step 1 — 投递任务
POST /api/internal/scheduler/tasks
{
  "task_type": "comment.moderate",
  "payload": { "comment_id": 42, "text": "..." }
}
Step 2 — Scheduler Center 调度
PENDING → RUNNING → 按 task_type 路由到 Comment Agent
Step 3 — Agent 执行
POST /api/internal/agent/run → Comment Agent
→ 审核评论内容
→ 写入审核结果到 Shared Memory
Step 4 — 结果聚合
Scheduler 收集执行结果
{
  "status": "succeeded",
  "result": { "approved": true, "score": 0.92 }
}
Step 5 — Shared Memory 沉淀
mempool.set("audit:comment:42", { "approved": true, "score": 0.92 })
→ 后续任务可通过 mempool.get() 复用
